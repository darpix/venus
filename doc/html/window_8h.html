◆ venus_initialize()

int venus_initialize ( )

Initializes venus and the libraries used.

This function does a couple of important things. First it initializes zlog, the logging library I have chosen to use. zlog needs to be started first in order to start logging immediately. As well as starting zlog, it also creates a connection to an X Server. It should be noted that this does not initalize OpenGL.

Returns
Returns whether it was successful or not
