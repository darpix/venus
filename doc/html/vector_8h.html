Defines a new vector type and includes function prototypes.

In Venus, a vector represents a mathematical vector, not a container like std::vector. The reason for this is that since Venus is a GUI/graphics library, it needs to manipulate points in two-dimensional space and three-dimensional space. This requires functions that can add, multiply, etc., vectors.

Parameters
TYPEThe datatype the vector should contain
NAMEThe name of the new vector
